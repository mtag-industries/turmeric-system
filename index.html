MTAG System Setup

One-time configuration — takes less than 2 minutes

Step 1: Open script.google.com → Create a new project named MTAG_Backend
Step 2: Paste the full contents of Code.gs into the editor. Run initializeSheets() once (grant permissions).
Step 3: Click Deploy → New Deployment → Type: Web App → Execute as: Me → Who has access: Anyone → Deploy. Copy the Web App URL.
